About Fledit

Fledit is the quickest way to stire and publish JSON data, no wonder if you are a human or a machine. If you need a simple bin to share piece of data or if you want to post/update information from your webapp, Fledit is the perfect tool.

Focussing on simplicity, it gives you access to your file from an API and from a WebSocket server that allows you to follow real-time updates.

No need to signup: Fledit will let you update your files as long as you use the same web browser. An admin link is also provided if you need to grant someone with writing abilities.
